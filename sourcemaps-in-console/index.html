Hello Goodbye

This page is a demonstration of source map support in the Firefox Web Console.

It contains a textbox, two radio buttons labeled "Hello" and "Goodbye", a button labeled "Greet", and a JavaScript source. When you press the button, the JavaScript source takes the contents of the textbox and logs a string to the Web Console using console.log(). The string is "Hello [contents]" or "Goodbye [contents]", depending on which radio button is checked.

We wrote the original source using CoffeeScript, and compiled it to JavaScript. We've also generated a source map for it.

  • See the original CoffeeScript source.
  • See the compiled JavaScript source.
  • See the source map.

The message that appears in the console has a link back to the source file. Without source map support, this will link to the compiled JavaScript source. With source map support, it will link to the original CoffeeScript source.

To try it out:

  • Make sure you're using Firefox version 50 or higher.
  • Visit "about:config", and check the value of the "devtools.sourcemap.locations.enabled" preference. If it's false, set it to true.
  • Open the Web Console.
  • Type a message in the textbox, and click the "Greet" button.

You should see a message in the Web Console, and the link it contains should be to the original source "main.coffee".
